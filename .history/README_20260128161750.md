# PyTorch Workflows

A comprehensive project for learning and implementing PyTorch workflows, covering fundamental concepts to computer vision pipelines.

## Getting Started

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Activate virtual environment (if using):
```bash
source .venv/bin/activate
```

## Project Structure

### cv-pipeline/
Computer Vision pipeline implementation with MNIST classifier
- `main.py` - Main training script
- `mnist_classifier.ipynb` - Interactive notebook for MNIST classification
- `helper_utils.py` - Utility functions for CV tasks
- `requirements.txt` - CV-specific dependencies
- `data/` - MNIST dataset storage

### dataManagement/
Data management and preprocessing workflows
- `C1_M3_Lab_data_management.ipynb` - Lab notebook for data management techniques
- `helper_utils.py` - Data management utilities

### simpleSteps/
Foundational PyTorch concepts and examples
- `tensors.ipynb` - Introduction to PyTorch tensors
- `simple_nn.ipynb` - Simple neural network implementation
- `activation_functions.ipynb` - Exploring activation functions
- `helper_utils.py` - General utility functions
- `helper_utils_act.py` - Activation function utilities
- `data.csv` - Sample data file

## Requirements

- Python 3.8+
- PyTorch 2.0+
- torchvision
- Additional dependencies in requirements.txt files
